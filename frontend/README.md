# ReciclaAlegre - React (frontend/react-app)


## Como rodar


1. `cd frontend/react-app`
2. `npm install`
3. `npm run dev`


O Vite vai abrir o dev server em `http://localhost:5173` por padrão.


## Conexão com backend


Configure a variável `VITE_API_BASE` no `.env` para apontar ao backend (ex: `VITE_API_BASE=http://localhost:8080`).


## Observações


- Os componentes e páginas são básicos — prontos para serem estilizados e integrados ao backend.
- Se quiser, eu gero os arquivos prontos numa .zip para você baixar e colar na pasta `frontend/react-app`.
